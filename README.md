Omnifood — Never cook again!
Omnifood is a premium food delivery service that uses AI to provide customized meal plans based on your health goals and preferences. This project is a front-end landing page built to showcase modern web design and responsive layouts.

🚀 Features
Omniscient AI: Custom meal plans tailored to your needs.

Sustainable & Local: 100% organic and fair-trade ingredients.

Responsive Design: Fully optimized for desktop, tablet, and mobile.

Modern UI: Built with clean typography and a vibrant color palette.

🛠️ Tech Stack
HTML5

CSS3 (Flexbox and CSS Grid)

JavaScript (Interactivity and smooth scrolling)

📁 Project Structure
/css: Stylesheets for layout and components.

/img: Optimized images and icons.

/js: Script files for navigation and animations.

index.html: The main entry point.
